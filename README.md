# AI-ML-PROJECT

## 📌 Project Overview

This project applies machine learning techniques to analyze patient drug reviews and predict overall satisfaction ratings based on features such as effectiveness, side effects, medical condition, and written review text.

The goal is to support better decision-making in healthcare by identifying patterns in patient experiences and enabling earlier prediction of drug satisfaction outcomes.

## 🎯 Problem Statement

The objective of this project is to predict a patient’s overall satisfaction rating for a drug using available features such as:

- Effectiveness rating
- Side effects rating
- Medical condition
- Written reviews (benefits, side effects, comments)

This model can help healthcare professionals and pharmaceutical companies better understand patient responses to medications and improve treatment decisions.

## 📊 Dataset Information
- Source: Drug review dataset (drugLibTrain_raw.tsv)
- Size: 4,000+ patient reviews
- Format: Tab-separated values (TSV)
- Type: Structured + Text data
  
## Features include:
- rating → Overall satisfaction (target variable)
- effectiveness → Effectiveness score
- sideEffects → Side effect severity score
- condition → Medical condition
- benefitsReview → Patient-written text
- sideEffectsReview → Patient-written text
- commentsReview → General review text
